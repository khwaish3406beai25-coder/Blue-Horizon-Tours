# 🌊 Blue Horizon Tours

A responsive multi-page travel website built using HTML, CSS and Vanilla JavaScript.

## 📄 Pages
- **index.html** – Home page
- **booking.html** – Trip booking with real-time price calculator & form validation
- **moodquiz.html** – Travel mood quiz with result based on answers
- **moodscale.html** – Interactive mood scale (Relaxed → Adventurous)
- **trip-planner.html** – Plan your trip
- **packing.html** – Packing checklist
- **maps.html** – Travel maps
- **gallery.html** – Photo gallery
- **friends.html** – Friends section
- **contact.html** – Contact form
- **login.html** – Login page

## ⚙️ Features
- 🧠 Mood Quiz with result logic
- 📊 Interactive Mood Scale with data attributes
- 💰 Real-time price calculator
- ✅ Form validation with error messages
- 💾 LocalStorage for booking data
- ⏱️ 24-hour offer countdown timer
- 🌙 Dark mode toggle
- ⬆️ Back to top button
- 📱 Responsive design

## 🛠️ Tech Stack
HTML | CSS | Vanilla JavaScript
